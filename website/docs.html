KasturiChain Documentation

Build sovereign, regenerative dApps on the world's first Vedic Layer-1 Blockchain.

1. Quickstart

To interact with KasturiChain from your web application or backend, install the official SDK:

npm install kasturichain

2. Using the Node.js SDK

The `kasturichain` package allows you to connect to the mainnet RPC, generate wallets, and call Agni Bhasha smart contracts.

const { KasturiProvider, KasturiWallet } = require('kasturichain');

// Connect to Mainnet
const provider = new KasturiProvider("https://kasturisundari.xyz/samparka");

async function init() {
    const block = await provider.getBlockNumber();
    console.log("Current Block:", block);
}
init();

4. JSON-RPC API Endpoints

If you prefer raw HTTP requests, you can query the blockchain directly via POST requests to our RPC URL.

curl -X POST https://kasturisundari.xyz/samparka \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","method":"eth_blockNumber","params":[],"id":1}'
