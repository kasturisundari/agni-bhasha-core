**Agni Bhasha** is the sovereign programming language of the **Kasturi Sundari** Ecosystem. Built entirely on the 4,000-year-old mathematical grammar of Panini's Ashtadhyayi, it serves as the native engine for KasturiChain. 

It breaks free from the legacy of western compilers and paradigms, offering a completely independent technological ecosystem: from the Kasturi Sundari compiler all the way up to the distributed Aadhaara database.

## 🌟 Core Features

1. **Absolute Sovereignty:** The `Agni.exe` compiler is completely self-hosted. The compiler's source code is written in Agni Bhasha itself (`सङ्कलक.अग्नि`).
2. **Paninian Syntax:** Code is written in pure Devanagari script, utilizing Sanskrit Dhatus (roots) for operations instead of English keywords.
3. **Aadhaara Database:** Built-in native database engine featuring:
   - Native B-Tree Indexing (`वृक्ष_अन्वेषणम्.अग्नि`)
   - Raft-based Distributed Consensus (`सहमति_तन्त्रम्.अग्नि`)
   - Quantum-Lattice Cryptography (`क्वाण्टम्_सुरक्षा.अग्नि`)
   - Direct Memory Access Caching (`स्मृति_तन्त्रम्.अग्नि`)

## 🚀 Getting Started

You do not need Rust, Python, or C to run Agni Bhasha. The ecosystem is completely self-contained.

### 1. Download
Download `Agni.exe` and the `padma_bhasha` standard library from the official releases.

### 2. Compile and Run
To compile any `.अग्नि` script, simply pass it to the compiler via your terminal:

```powershell
# Set terminal to UTF-8 to support Devanagari
chcp 65001

# Run the Aadhaara Database Dashboard
.\Agni.exe "padma_bhasha\अनुप्रयोग\आधार_नियन्त्रण_कूप.अग्नि"
```

## 🏗️ Architecture

The ecosystem consists of the following sovereign components:
*   `Agni.exe`: The master compiler/VM.
*   `padma_bhasha\सङ्कलक\`: The self-hosted compiler source code.
*   `padma_bhasha\मानक\`: The Standard Library (File I/O, Networking, DMA).
*   `padma_bhasha\आधार_दत्तांश\`: The Distributed Database Engine.
